# Менеджер игры «Что за мем»

Небольшое настольное приложение под Windows для проведения игры «Что за мем»:

- банк ситуаций для мемов (можно пополнять и редактировать);
- конструктор текста программы игры в стиле вашего примера;
- таймер раунда (по умолчанию 3 минуты) с визуальным отсчётом и звуковым сигналом.

## Как запустить

1. Установите Python 3.10+ для Windows с сайта `https://www.python.org/` и при установке поставьте галочку **Add Python to PATH**.
2. Скачайте/скопируйте файлы проекта в одну папку (например, `d:\WAM`).
3. Откройте PowerShell/командную строку в этой папке.
4. Установите зависимости (фактически используется стандартный Tkinter, но команда не повредит):

```bash
pip install -r requirements.txt
```

5. Запустите приложение:

```bash
python mem_game_manager.py
```

## Как пользоваться

- **Банк ситуаций**: добавляйте, редактируйте и удаляйте ситуации. Все они хранятся в файле `situations.json` рядом с программой.
- **Программа игры**: задайте время старта, тему/стиль и количество ситуаций, нажмите «Сгенерировать программу игры», затем при необходимости скопируйте текст в буфер и вставьте в чат.
- **Таймер**: настройте длительность раунда (по умолчанию 180 секунд) и нажмите «Старт» сразу после отправки поста с ситуацией. По окончании будет звуковой сигнал и всплывающее окно.
